# Softlend Rule Engine (Role 3)

## How to run tests
1. Navigate to the `rule_engine/` folder.
2. Install PyYAML: `pip install pyyaml`
3. Run the tests: `python test_cases.py`

## How to add a new rule
1. Open `rules.yaml`.
2. Add a new entry under `gap_rules` or `eligibility_rules`.
3. No code changes required in `engine.py`.
